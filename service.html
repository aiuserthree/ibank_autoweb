AI가 자율 수행하는
웹사이트 제작 전 과정

챗봇 상담부터 서버 배포까지, 기존 에이전시의 모든 과정을 AI 에이전트가 자동으로 처리합니다.

10단계 워크플로우

단계별 상세 프로세스

각 단계가 AI 에이전트에 의해 자율적으로 실행됩니다.

💬
⏱ ~30분

1단계: 챗봇 상담

AI 챗봇이 프로젝트 요구사항을 수집합니다. 업종, 목적, 레퍼런스, 예산, 일정 등을 대화 형식으로 파악하고 즉시 초기 방향을 제시합니다.

  • 자연어 대화 기반 요구사항 수집
  • 레퍼런스 URL 분석 자동화
  • 초기 방향 즉시 제안
🔍
⏱ ~1시간

2단계: 자동 리서치

입력된 레퍼런스 사이트와 경쟁사를 AI가 자동으로 분석합니다. 업계 트렌드, 디자인 패턴, 기능 구성을 파악해 최적의 기획 방향을 도출합니다.

  • 경쟁사 사이트 30개 자동 분석
  • 업계 디자인 트렌드 파악
  • 기술 스택 추천
💰
⏱ 즉시

3단계: 견적 생성

리서치 결과와 요구사항을 바탕으로 정확한 견적을 자동 생성합니다. 페이지 수, 기능, 난이도를 고려한 투명한 항목별 견적서를 제공합니다.

  • AI 기반 정확한 견적 산출
  • 항목별 투명한 비용 공개
  • 3가지 옵션 견적 제안
📋
⏱ ~2시간

4단계: 제안서 자동 작성

브랜드 스타일, 기획 방향, 기술 스펙이 담긴 전문 제안서를 AI가 자동 작성합니다. PDF와 온라인 뷰어 형식으로 제공됩니다.

  • 전문 제안서 PDF 자동 생성
  • 브랜드 무드보드 포함
  • 기술 스펙 상세 명세
📨
⏱ 즉시

5단계: 자동 발송

완성된 제안서와 견적서를 이메일, 카카오톡으로 자동 발송합니다. 고객 확인 후 계약서도 전자서명으로 처리됩니다.

  • 이메일 + 카카오 동시 발송
  • 전자서명 계약서 연동
  • 계약 완료 즉시 착수
🎨
⏱ ~1일

6단계: 시안 생성

AI 디자이너가 브랜드 아이덴티티에 맞는 UI 시안을 생성합니다. 3가지 방향의 시안을 제시하고 고객이 선택합니다.

  • AI 기반 3가지 디자인 방향 제시
  • 브랜드 컬러·폰트 자동 추출
  • 실시간 디자인 피드백 반영
⚙️
⏱ 3~7일

7단계: 자동 개발

선택된 디자인을 기반으로 AI 개발 에이전트가 코드를 자동 생성합니다. Next.js, React 기반의 퍼포먼스 최적화된 코드를 생성합니다.

  • Next.js 기반 자동 코드 생성
  • 반응형 레이아웃 자동 적용
  • SEO 최적화 자동 처리
🔄
⏱ ~1일

8단계: 공유 및 피드백

개발된 결과물을 스테이징 URL로 고객과 공유합니다. 피드백을 수집하고 AI가 즉시 수정 사항을 반영합니다.

  • 스테이징 환경 즉시 제공
  • AI 피드백 자동 분류·처리
  • 실시간 수정 현황 공유
✅
⏱ ~1일

9단계: AI 검수

SEO, 성능, 보안, 접근성 4개 영역을 AI가 자동으로 검수합니다. 100점 만점 기준으로 점수를 측정하고 부족한 부분을 자동 개선합니다.

  • 30개 이상 자동 체크리스트
  • 성능 점수 90+ 목표
  • 보안 취약점 자동 패치
🚀
⏱ 즉시

10단계: 자동 배포

검수 완료 후 NCP(Naver Cloud Platform) 또는 고객 서버에 자동 배포합니다. SSL, CDN, 모니터링이 자동으로 설정됩니다.

  • 원클릭 자동 배포
  • SSL 인증서 자동 발급
  • 24시간 업타임 모니터링 시작
